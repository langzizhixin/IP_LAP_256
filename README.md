| Capability | LangQing |
|---|---|
| Fast Response Latency | <500ms |
| Normal Conversation Latency | ~1.5s |
| Lip Sync Similarity | Up to 96% |
| Realtime Gesture Sync | Yes |
| Seamless Video Switching | Yes |
| CPU Realtime Inference | Supported |
| GPU Concurrent Sessions | 4~6 streams on RTX 3060 |
| WebRTC | Supported |
| Local Deployment | Supported |
| Cloud Deployment | Supported |
| Multi-language | Supported |
| RAG Workflow | Supported |
